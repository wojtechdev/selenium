## üöÄ Getting Started

### Prerequisites

- Node.js (v24 or higher)
- Chrome browser
- ChromeDriver (automatically managed by Selenium)

### Installation

```bash
npm install
```

### Run Tests

Run all tests in parallel and generate HTML report:

```bash
npm test
```

Build TypeScript files:

```bash
npm run build
```

Run tests in watch mode:

```bash
npm run test:watch
```

View the test report at: `reports/report.html`
